# ⚡ Quick Reference

Quick commands and shortcuts for ProxyMukt.

---

## 🚀 Development

### Start Project
```bash
# Backend
cd server && npm run dev

# Frontend
cd client && npm run dev
```

### Seed Database
```bash
cd server && npm run seed
```
